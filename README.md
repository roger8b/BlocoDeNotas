## Bloco de notas

---

### Objetivo:

Objetivo deste aplicativo é fornecer ao usuário uma maneira simples de guardar anotações.

Demonstrando o uso do banco de dados SQLite para persistência de dados no android.

---

### Dados da aplicação

- Linguagem: Java
- SDK: 28
- Min SDK: 23
